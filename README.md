# Linkeefy

Linkeefy is a React + Firebase app that allows you to share your favorite links (social media, personal links, etc).

## Technologies used:

- React
- Firebase (Firestore, Auth, Storage)
- TailwindCSS
- React Router v6
- Zustand
- Styled components
